Each method contract MUST check `RunEventStore.hasTerminalEvent(run.runId())` or equivalent before publishing through `EventSink`, so queued cancellation, dispatcher timeout, dispatcher failure, and cancellation detected outside runtime terminal emission create exactly one durable terminal RunEvent.
